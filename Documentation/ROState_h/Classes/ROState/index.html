An alternate next state in the tree, if the tree forks. May be nil.

matchingCharacter

@property (strong,
    nonatomic) NSString* matchingCharacter;
Discussion

If alternateState=nil, we have "any character" match, i.e. we match and move to nextState. If alternateState!=nil, we have a fork in the tree, i.e. we don't match and move to nextState and alternateState.
